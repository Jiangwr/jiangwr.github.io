记录二分的几种常用模版。


一种是找重复元素的左右边界，以 LeetCode34 的代码为例。

```
    public int[] searchRange(int[] nums, int target) {
        if (nums == null || nums.length == 0) {
            return new int[]{-1,-1};
        }
        int left = findLeftIdx(nums, target);
        if (nums[left] != target) {
            return new int[]{-1, -1};
        }
        int right = findRightIdx(nums, target);
        return new int[]{left, right};
    }

    private int findLeftIdx(int[] nums, int target) {
        int low = 0, high = nums.length - 1;
        while (low < high) {
            int mid = low + high >> 1;
            if (nums[mid] >= target) {
                high = mid;
            } else {
                low = mid + 1;
            }
        }
        return low;
    }

    private int findRightIdx(int[] nums, int target) {
        int low = 0, high = nums.length - 1;
        while (low < high) {
            int mid = low + high + 1 >> 1;
            if (nums[mid] <= target) {
                low = mid;
            } else {
                high = mid - 1;
            }
        }
        return low;
    }
```


还有一种是找插入位置，以 LeetCode35 的代码为例

```
    public int searchInsert(int[] nums, int target) {
        int low = 0, high = nums.length - 1;
        while (low <= high) {
            int mid = low + high >> 1;
            if (target == nums[mid]) {
                return mid;
            } else if (nums[mid] > target) {
                high = mid - 1;
            } else {
                low = mid + 1;
            }
        }
        return low;
    }
```
